Casting a Double to an Int

(int)

You might have an int variable in your code. After running math on some doubles, you want the final answer to be rounded down to the nearest integer. In this case, you would cast the final answer to an int this way.

int anInteger;

double aDouble;

[some code here]

anInteger = (int)aDouble

Note that if the double holds the value 3.9999, when it is cast to an integer, it will hold the value 3. Everything after the decimal point is tossed. This is called flooring a number.

(double)

You may want to do the opposite. Perhaps if you are dividing integers and you don't want to lose the remainder. You may cast an int to a double before doing the operation.

System.out.println("3/4 = " + 3/4);

System.out.println("(double)3/4 = " + (double)3/4);

The output for the above statements will be:

3/4 = 0

(double)3/4 = .75

It may appear that (double)3 divided by 4 is incompatable, but java will cast the other integer to a double before performing the operation.

String.valueOf()

I was curious if I could cast an int or a double to a String. This gives an "incompatible types" error. However, I leaned that you can convert an integer or double to a string using String.valueOf() in this manner:

String aString;

double aDouble = 36.81;

aString = String.valueOf(aDouble);

aString will then have the value "36.81".
