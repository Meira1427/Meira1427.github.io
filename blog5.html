Importing Scanner Class

import java.util.Scanner

This is going to be a bit of a hack answer because I don't fully understand the concepts. Please bear with me.

In Java, you don't have to write all of your code from scratch. You can use tools already written in the Java library. Within the Java library, there is a utility (.util) library. Within the utility library, a Scanner class and associated methods. The Scanner class has the ability to read from the keyboard or from a file and put data it finds there into Strings, integers, doubles, floats, and booleans. So you import this class from the utility library if you intend the ask the user to give you data from the keyboard or if you wish to read data from a file.

import java.util.Scanner

Scanner myKeyboard = new Scanner(System.in);

If you do this, you have a new variable called myKeyboard of type Scanner class which reads input from System.in, which is the user's keyboard.

System.out.print("Enter your name: ")

String userName = myKeyboard.next();

This will pull the first word the user enters, and put it in the new variable called userName. If you'd like them to be able to put in a first and last name, use nextLine:

String userName = myKeyboard.nextLine();
